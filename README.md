# IITK-IESM: Integrated Energy Systems Model for India

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.11+](https://img.shields.io/badge/python-3.11+-blue.svg)](https://www.python.org/)

IITK-IESM is an open-source modelling suite for exploring India's energy transition pathways to 2070. It orchestrates:

- **OSeMOSYS** for long-term, multi-sector capacity expansion.
- **PyPSA** for high-resolution, hourly grid and sector-coupling validation.

Developed at the Indian Institute of Technology Kanpur, this tool incorporates novel features like battery degradation, critical mineral constraints, and myopic foresight to provide realistic, policy-relevant insights.
